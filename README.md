# AI Marketing Agent (Starter)

Minimal Next.js (App Router) + Tailwind project including:
- `/` hero
- `/flow` chat onboarding
- `/plan` strategy result
- `/connect` checklist
- `/dashboard-demo` sample analytics
- `/pricing` and `/legal` stubs

## Run
1. Install Node 18+
2. `npm install`
3. `npm run dev`
4. Open http://localhost:3000
